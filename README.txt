Upload these contents to a public GitHub repository for GitHub Pages.

Files:
- index.html  -> your website homepage

Quick publish steps:
1. Create a new PUBLIC repository on GitHub.
2. Upload index.html to the root of the repository.
3. Go to Settings > Pages.
4. Under Build and deployment, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
5. Save.
6. Wait a minute, then open the GitHub Pages URL shown there.

Notes:
- This version uses an inline SVG logo, so no separate logo file is required.
- If you later add images or PDFs, upload those too and keep paths relative.
